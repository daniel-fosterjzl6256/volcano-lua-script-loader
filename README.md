## Overview

Volcano PC is a Windows-focused Roblox script executor for injecting Lua scripts into active game sessions. It supports custom gameplay changes, automation workflows, and scripting experiments. The application emphasizes a quick workflow: injection is available from a single control and is intended to begin almost immediately, without complicated menus or extensive setup.

The included script hub extends the tool beyond basic injection. It organizes more than 500 community-verified scripts into searchable categories, covering GUI utilities, game-specific modifications, and automation tools. A persistent execution queue also lets users prepare several scripts and run them consecutively without starting each one by hand.

## Frequently Asked Questions

**Is Volcano PC safe to use?**  
Volcano PC is supplied as-is for educational and experimental use. You assume the risk when using it. The developers do not guarantee that the software will avoid detection or prevent account actions.

**Does it support the newest Roblox release?**  
Its update system is intended to preserve compatibility with the latest Roblox builds. A Roblox update can nevertheless interrupt injection temporarily until a corresponding patch becomes available.

**What distinguishes Volcano PC from other executors?**  
The project is built around a lightweight workflow that requires no key and includes its own script hub. There are no premium levels or key systems, so the application can be used immediately after downloading.

**Could using Volcano PC result in a ban?**  
Third-party executors are against the Roblox Terms of Service, and account bans may occur. Roblox account actions are outside the developers' responsibility.

**Where does Volcano PC save scripts?**  
Scripts imported by the user are kept in `scripts/user/`, while cached hub content is placed in `scripts/hub/`. Either location can be backed up or moved manually.
